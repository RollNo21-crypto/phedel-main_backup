Technical Specifications

Detailed specifications for wall mount network cabinets

Physical Dimensions

Height Options: 6U, 9U, 12U, 15U
Width: 600mm (19\")
Depth Options: 300mm, 450mm, 600mm
Load Capacity: 60kg (132 lbs)
Material: Cold-rolled steel, 1.5mm thickness
Finish: RAL 7035 light gray powder coating

Features & Standards

Protection Rating: IP20
Mounting: Wall bracket included
Door Type: Tempered glass or perforated steel
Locking: 3-point locking system
Ventilation: Top and bottom ventilation panels
Standards: IEC 60297, EIA-310-D

Key Features

Why choose our wall mount network cabinets

Space Saving Design

Compact wall-mounted design maximizes floor space while providing secure equipment housing for network infrastructure.

Cable Management

Integrated cable management system with multiple entry points and routing channels for organized installations.

Easy Installation

Pre-drilled mounting holes and included wall brackets ensure quick and secure installation on various wall types.

Visual Access

Optional tempered glass door provides visual access to equipment status lights and indicators without opening the cabinet.

Security Features

3-point locking system with cylinder lock provides secure access control and equipment protection.

Ventilation System

Optimized ventilation design with top and bottom panels ensures proper airflow for equipment cooling.
